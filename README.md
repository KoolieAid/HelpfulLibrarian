# Helpful Librarian
Helpful Librarian is a 2D single player Filipino reading comprehension educational puzzle game. In this game, you play as a librarian who helps countless visitors find the books that they want to read. You need to look for words and visual cues in books to match the visitorâ€™s request. Help as many visitors as quickly as you can before their patience runs thin.
